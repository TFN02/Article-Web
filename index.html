About

ditulis oleh Tegarfarisnurhakim,pada 25 februari 2021.

Hi! Perkenalkan nama saya Tegar Faris Nurhakim saya adalah seorang mahasiswa program studi Teknik Informatika di Politeknik Negeri bandung. Umur saya 18 tahun dan saya tinggal bersama orang tua saya di Kabupaten Garut. Saya sangat menyukai dunia IT karena dari kecil saya sudah dikenalkan dengan yang namanya teknologi. Dengan teknologi saya dapat menemukan hal hal baru yang saya belum ketahui.


Dengan dasar kesukaan saya terhadap dunia IT saya memutuskan untuk memilih program studi D3-Teknik Informatika. Karena di program studi ini saya dapat mengetahui cara pembuatan suatu aplikasi maupun website. Saya memiliki ketertarikan terhadap pembuatan website karena pada saat pembuatan website saya dapat menuangkan isi pikiran atau ide ide yang ada pada otak saya. Oleh karena itu saya menekuni hal hal yang menyangkut tentang web programming. dan alhamdulillah saat ini saya menguasai HTML, CSS
